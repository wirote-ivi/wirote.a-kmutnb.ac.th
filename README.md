Cyclone IV FPGA SystemVerilog Projects
Educational SystemVerilog projects for Cyclone IV FPGA.
## Hardware
- FPGA Board : Cyclone IV
- Device     : EP4CE6E22
- Software   : Quartus Prime
- Language   : SystemVerilog

## PProject Purpose

This project is used to verify the FPGA board functionality and the
computer toolchain environment including Quartus compilation and
SystemLink connectivity.
  Toolchain_Test Quartus and Cyclone IV (EP4CE6E22)
  │
  ├── LED_Test
  ├── Button_Test
  ├── Counter_Test
  ├── FSM_Test
  └── SystemLink_Test
